## Impeccable (structure and workflow)

The repository structure — one main skill, a set of command skills, per-command `SKILL.md` files, a shared set of reference material, and the Claude Code plugin/marketplace wiring — is modeled after [impeccable](https://github.com/pbakaus/impeccable) by Paul Bakaus.

## e18e initiative

All domain content — the cleanup / speedup / levelup pillars, the module-replacements and native-replacements catalogs, the ecosystem-issues workflow, and the recommended tooling — is adapted from the [e18e initiative](https://e18e.dev) and its associated repositories:

- [e18e.dev](https://e18e.dev) — docs and guides
- [e18e organization](https://github.com/e18e) — CLI, ESLint plugin, framework-tracker, ecosystem-issues
- [es-tooling/module-replacements](https://github.com/es-tooling/module-replacements) — the curated replacement manifests

e18e is a community initiative; this project is not officially affiliated.
